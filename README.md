
# Peblo AI Notes Workspace

## Stack
- React + Vite
- Node.js + Express
- MongoDB
- JWT Authentication

## Features
- User Login/Signup
- Notes CRUD
- Protected Routes
- MongoDB Integration

## Backend Setup

```bash
cd backend
npm install
npm run dev
```

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

## Environment Variables

Create `.env` inside backend:

```env
PORT=5000
MONGO_URI=your_uri
JWT_SECRET=your_secret
```
